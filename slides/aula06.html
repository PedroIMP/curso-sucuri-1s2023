O que é um módulo?

De forma geral, um módulo é um arquivo:

  • que possui código Python
  • e que pode ser utilizado (importado) em outros programas ou módulos

Como Python encontra o módulo poligonos?

Como Python encontra o módulo poligonos?

A resolução é feita nesta ordem:

  1. Primeiro procura por um módulo built-in com o nome "polígono" (não é nosso caso);
  2. Depois ele procura por um arquivo nomeado "poligono.py" na lista de diretórios definida em sys.path.

Como Python encontra o módulo poligonos?

...e Python inicializa a variável sys.path com o seguinte:

  1. Diretório contendo o arquivo de entrada ("main.py" no nosso exemplo);
    Por isso que o arquivo poligonos.py é encontrado!
  2. Caminhos presentes na variável de ambiente PYTHONPATH;
  3. Caminhos específicos da instalação utilizada.

Utilizando a função built-in open()

f = open(caminho, modo)

O retorno é um objeto cujos métodos são responsáveis por ler e escrever em arquivos.

f = open(caminho, modo)

caminho: o caminho do arquivo a ser aberto

Se o caminho for relativo, o current working directory será utilizado como base para resolução.

Pode ser um str ou um objeto to tipo pathlib.Path.

O que é um caminho?
f = open(caminho, modo)

modo: o modo de abertura

É uma string contendo combinação de caracteres para definir o modo de abertura.
